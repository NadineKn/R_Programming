# Individuell inlämningsuppgift – R-programmering för dataanalys

Försäkringskostnader - Det här projektet analyserar vilka faktorer hänger ihop med försäkringskostnader.

## Filer

-   `scripts/01_load_data.R` läser in data
-   `scripts/02_prepare_data.R` städar och förberedar data
-   `scripts/03_analysis.R` gör analysen
-   `scripts/04_figures.R` skapar och sparar figurer
-   `run_analysis.R` kör hele projektet

## Så kör du projektet

Öppna projektet i RStudio och kör `run_analysis.R`

## Paket

-   tidyverse

## Output

Figurer sparas i `figures/`.
